หน่วยการเรียนรู้ที่ 3 การใช้แท็กของภาษา HTML ในการสร้างเว็บไซต์


วิธีการใช้แท็กในเว็บไซต์

1. พิมพ์ h1 เพื่อเป็นการประกาศใช้แท็ก h1 หรือแท็กหัวข้อ

2. แสดงผลลัพธ์ของแท็ก h1-h6 โดยเรียงตามความหนาของ h1 ลงมา

3. พิมพ์ img เพื่อเป็นการประกาศใช้แท็กรูปภาพ

4. ทำการเปิดโฟลเดอร์ที่อยู่ของรูปภาพแล้วทำการลากมายังโปรแกรม VS Code

5. ทำการคัดลอกที่อยู่ไฟล์โดยทำตามภาพ

6. นำมาวางตามภาพ

7. กำหนดความกว้าง และความสูงของภาพ ตามภาพ

8. แสดงผลลัพธ์อของแท็ก img

9. ประกาศใช้แท็ก a โดยพิมพ์ a และกด Enter บนแป้นพิม์

10. ทำการคัดลอกหน้าเว็บที่ต้องการให้ไปแสดง

11. นำที่อยู่เว็บมาวางตามภาพ

12. แสดงผลลัพธ์ของแท็ก a หรือแท็กลิงค์ไหน้าอื่น

13. ประกาศใช้แท็ก br หรือแท็กเว้นบรรทัด

14. ผลลัพธ์ของแท็ก br จะทำการขึ้นบรรทัดใหม่ให้กับตัวที่ถูกกำหนดให้ขึ้นบรรทัดใหม่ br

15. ประกาศใช้แท็ก p หรือแท็กเนื้อหา

16. กำหนดข้อมูลในแท็กต่างๆใน p

17. ประกาศใช้ dd แท็ก dd เป็นแท็กกำหนดย่อหน้าของบรรทัด

18. แสดงผลลัพธ์ของแท็ก dd และแท็ p


แท็กที่ใช้ในการสร้างเว็บ

h1 = แท็กหัวข้อ

img = แท็กรูปภาพ

a = แท็กลิงค์ไปหน้าอื่น

p = แท็กกำหนดเนื้อหา

dd = แท็กกำหนดย่อหน้า

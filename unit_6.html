8. รอดาวโหลดให้เสร็จสิ้น

9. ระหว่างรอดาวโหลด ให้ไปติดตั้ง Node.js ที่เว็บไซต์ของ Node.js

10. เลือกดาวโหลดอันแรกที่เป็นเวอร์ชั่นที่เสถียรที่สุด เมื่อดาวโหลดเสร็จแล้วให้ติดตั้งโปรแกรม

15. คลิกที่ View เลือกที่ Terminal

16 . จะปรากฏคำสั่งตามภาพ

17. ไปคัดลอกคำสั่งที่เว็บไซต์ Firebase ตามภาพ

18. นำคำสั่งที่คัดลอกไปวางตามภาพจากนั้นกด Enter

19. หลังจากคำสั่งติดตั้งเสร็จสิ้นแล้วจพปรากฏคำสั่งใหม่

22. จะปรากฏผลลัพธ์ตามภาพ

23. คัดลอกคำสั่งที่ 3

24. และนำไปวางยังโปรแกรมตามภาพ และกด Enter

25. จะปรากดฏผลลัพธ์ตามภาพ และพิมพ์ Y

26. เลือกลูกศรชี้ลงบนแป้นพิมพ์และมาที่ Hosting: configure files for Fierbase Hosting and (optionally) set up GitHub Action deploys

27. กด Spacebar บนแป้นพิมพ์ เพื่อเลือก และกด Enter

28. เลือก Use an existing project และกด Enter

29. เลือก ชื่อโปรเจกต์ที่สร้างไว้ และกด Enter

30. พิมพ์ y บนแป้นพิมพ์

เว็บไซต์ (Web Site)

เว็บไซต์ (Web Site) คือที่อยู่หลักของการเก็บรวมรวมข้อมูลต่างๆ ที่เราต้องการนำไปเผยแพร่สู่อินเตอร์เน็ต เพื่อเปิดให้ผู้ใช้ที่สนในเนื้อหาของเว็บไต์ของเราเข้าไปอ่านหรือรับชมเว็บไซต์ได้ เว็บซ์ที่สร้างกันมีอยู่หลายประเภท เช่น เว็บไซต์ส่วนตัว เว็บไซต์หน่วยงาน เว็บไซต์องค์กรณ์ หรือเว็บไซต์ของบริษัทเอกชนที่ดำเนินงานด้านธุรกิจหรือด้านอื่นๆ เป็นต้น โดยภาษาที่จะใช้ในการเขียนเว็บไซต์ ในบทเรียนนี้จะมีอยู่ด้วยกัน 2 ภาษาดังนี้


1. ภาษา HTML (Hypertext Markup Language)

ภาษา HTML หรือ Hypertext Markup Language มีไฟล์นามสกุลชื่อ .html หรือ htm เป็นภาษาที่ทำหน้าที่ในการสร้างเว็บเพจ (web Page) หรือสร้างส่วนแสดงผลในเบราเซอร์ แล้วนำเว็บเพจเหล่านั้นมาเชื่อมโยงเข้าด้วยกัน รวมกันเป็นเว็บไซต์ต่างๆ เราเรียกเว็บไซต์ที่เกิดมาจาก HTML เพียงอย่างเดียวว่า Static Web Page

ในยุคเริ่มต้นภาษา HTML ถูกสร้างขึ้นมาเพื่อทำหน้าที่สร้างส่วนแสดงผลต่างๆ ในหน้าเว็บไซต์เพียง อย่างเดียว แต่ในยุคปัจจุบัน ภาษา HTML ถูกนำไปใช้พัฒนา Desktop Apps และ Mobile Apps ผ่านการใช้งาน Libary หรือ Freamwork ต่างๆ ส่งผลให้ภาษา HTML ยังคงเป็นฐานความมั่นคงที่นักพัฒนามือใหม่ใช้เป็นก้าวแรกนำไปสู่การเขียนโปรแกรมด้านอื่นๆ ได้อีกด้วย


2. ภาษา CSS (Cascading Style Sheet)

ภาษา CSS หรือ Cascading Style Sheet คือตัวกำหนดรูปแบบและเสริมลูกเล่นในการแสดงข้อมูลและเนื้อหา ให้สวยงาม และน่าสนใจมากขึ้น โดยจะใช้ CSS กำหนดรูปแบบตัวอักษร สี เส้นขอบ และ CSS3 หรือ Cascading Style Sheet ver.3 ใน CSS3 มีการ เพิ่มเติมในส่วนของการใช้งาน Selectors เพิ่มมากยิ่งขึ้น ซึ่ง CSS2 ไม่สามารถทำได้ในหลายๆความสามารถในการแสดงผล ของ CSS3 และในปัจจุบันบราวเซอร์ต่างๆ ได้มีการรองรับ CSS3 แล้ว

CSS3 ยังมีส่วนช่วยในการเขียน CSS และ Responsive ง่ายขึ้น เสริมการใช้งานด้วย CSS Freamwork และปัจจุบันมีการพัฒนา CSS Freamwork ขึ้นมาเพื่อให้ใช้งานเพื่อการรเขียนโค้ดได้ง่ายขึ้น โดนมี Freamwork หลักที่เราคุ้นเคย เช่น jQuery UI CSS Bootstrap และ Freamwork Foundation สิ่งเหล่านี้จะช่วยให้การสร้างเว็บไซต์ได้หลากหลายมากขึ้น

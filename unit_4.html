วิธีการตกแต่งเว็บไซต์ด้วย CSS

1. ทำการพิมพ์ link ลงบนแป้นพิมพ์ เพื่อประกาศใช้คำสั่ง link

2. สร้างไฟล์ CSS

3. ทำการใส่ชื่อไฟล์ Style.CSS เพื่อทำการลิงค์ไฟล์

4. ทำการีเซ็ทหน้าเว็บไซต์โดยการเซ็ทค่า padding และ margin ให้มีค่าเป็น 0

5. แสดงผลลัพธ์ของการรีเซ็ทหน้าเว็บ

6. ตัวอย่างต้องการเปลี่ยนสีของหัวข้อ(แท็ก h1) ไประบุยังหน้า Style.CSS

7. ประกาศแท็ก ในหน้า Style.CSS และระบุค่าสีที่ต้องการตามภาพ

8. แสดงผลลัพธ์ของการประกาศแท็กและการระบุสี

9. ตัวอย่างต้องการนำแท็ก body ไปเปลี่ยนสีพื้นหลังของเว็บไซต์ ภายในหน้า Style.CSS

10. ค้นหาโค้ดสีที่ต้องการภายในเว็บไซต์ทั่วไป

11. ระบุแท็ก body และนำโค้ดสีที่ได้มาวางใน background: โค้ดสี;

12. แสดงผลลัพธ์ของการระบุโค้ดสีภายในหน้า Style.CSS

13. ตัวอย่างต้องการเปลี่ยนขนาดตัวอักษรของหัวข้อแท็ก h1 ให้ระบุ

14. แสดงผลลัพธ์ของการเพิ่มคำสั่ง font-size เพื่อเปลี่ยนตัวอักษร

หน่วยการเรียนรู้ที่ 5 การจัดหน้าเว็บไซต์ด้วย CSS3


วิธีการจัดหน้าเว็บไซต์ด้วย CSS

1. สร้างแท็ก (div)

2. กำหนดค่าให้ div เช่น (class="content") ตามภาพ

3. คัดลอก content

4. นำค่าไปประกาศยังหน้า Style.css โดยกำหนดเป็น .content

5. ประกาศใช้คำสั่งตามภาพ

6. ผลลัพธ์ของการกำหนดค่าภายในหน้า Style.css

7. ถัดมาเป็นการกำหนดระยะห่างของขอบภายใน (Padding)

8. แสดงผลลัพธ์ของการกำหนด (Padding) ขอบทุกมุมจะห่างจากพื้นหลังสีขาว 40px

10. ย้ายแท็กหัวเรื่องไว้เหนือรูปภาพ

11. กำหนดขนาดตัวอักษร

12. ถัดมาเป็นการลบเส้นใต้ของข้อความลิงค์ (a)

13. กำหนดแท็ก a และใส่คำสั่งลบเส้นใต้ text-deccoration: none;

14. แสดงผลลัพธ์ของการประกาศใช้คำสั่งลบเส้นใต้

15. หากต้องการใส่ข้อความด้านหน้าแท็กลิงค์ (แท็ก a ) ให้ทำการสร้างแท็ก p และใส่ข้อความที่ต้องการได้ทันที

16. ข้อความจะถูกแสดงผลด้านหน้าข้อความลิงค์ (แท็ก a)

17. การกำหนดสีข้อความลิงค์ ใช้คำสั่ง (color: ระบุสีที่ต้องการ;) ข้อความลิงค์บนหน้าเว็บจะถูกเปลี่ยนเป็นสีที่กำหนดไว้

18. การสร้างหัวข้อใหม่

19. แสดงผลลัพธ์ของการกำหนดหัวข้อใหม่

20. การกำหนดลักษณะเฉพาะหรือการกำหนดค่าเพียงส่วนเดียวกรือบางส่วน กรณีนี้ต้องการเปลี่ยนขนาดตัวอักษรของหัวข้อใหม่ โดยการกำหนดค่าภายในแท็กของหัวข้อใหม่ (h2) ใช้คำสั่ง (Style="font-size:18px;")

21. แสดงผลลัพธ์ของการใช้คำสั่งของข้อความ

22. ต้องการกำหนดให้แสดงหน้าต่างใหม่หลังจากคลิกที่ข้อความลิงค์ โดยการ

23. จะปรากฏผลลัพธ์ดังภาพ

24. กำหนดให้ภาพอยู่ตรงกลางหน้าเว็บโดยการใช้คำสั่ง Center คลุมยังแท็กรูปภาพดังภาพ

25. แสดงผลลัพธ์ของการใช้คำสั่ง

26. การกำหนดค่า id="ค่าที่ต้องการ" จะคล้ายกับการกำหนด class ="ค่าที่ต้องการ" กรณีจะแนะนำการกำหนดค่าวิธีที่ 2 โดยใช้ id="ค่าที่ต้องการ"

27. ประการใช้ค่าโดยกำหนด # ตามด้วยค่าที่กำหนดใน id เช่นดังตัวอย่าง id="Music"

28. ทดสอบโดยการให้ข้อความไปอยู่ด้านขวาของหน้าเว็บโดยการใช้คำสั่ง (text-align:right;)

29. ข้อความจะถูกย้ายมายังด้านขวาของเว็บตามที่กำหนด


แท็กและค่าที่ใช้กำหนด

div = เปรียบเสมือนกล่องเปล่ากล่องหนึ่งไว้ใส่องค์ประกอบต่างๆเพื่อที่จะนำไปตกแต่งยังหน้า CSS หรือนำไปทำการเปลี่ยนแปลงต่างๆ และความสามารถของ div คือ การนำไปครอบส่วนต่างๆของ HTML เพื่อนำไปกำหนดลักษณะให้มีรูปแบบเฉพาะส่วนตามที่เราต้องการ โดยมีวิธีการใส่ค่าของ div เพื่อนำไประบุยัง CSS มีดังนี้

id = "ใส่ข้อความที่ต้องการหนด"

class ="ใส่ข้อความที่ต้องการหนด"

style ="ระบุคำสั่งที่ต้องการกำหนดลักษณะ"
